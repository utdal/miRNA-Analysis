# Test Dataset

Four samples were retrieved from NCBI: SRA Study [SRP463256](https://trace.ncbi.nlm.nih.gov/Traces/index.html?view=study&acc=SRP463256), Bioproject [PRJNA1021141](https://www.ncbi.nlm.nih.gov/bioproject/PRJNA1021141).

Only the first 1,000,000 reads from each sample were included in these test datasets.
